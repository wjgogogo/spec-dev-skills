# XHS Selection Guide

## Auto Selection Rules

根据内容信号自动推荐 style 和 layout。

| Content Signals | Style | Layout |
|-----------------|-------|--------|
| Beauty, fashion, cute, girl, pink | `cute` | sparse/balanced |
| Health, nature, clean, fresh | `fresh` | balanced/flow |
| Life, story, emotion, warm | `warm` | balanced |
| Warning, important, critical | `bold` | list/comparison |
| Professional, business, elegant | `minimal` | sparse/balanced |
| Classic, vintage, traditional | `retro` | balanced |
| Fun, exciting, wow | `pop` | sparse/list |
| Knowledge, concept, productivity | `notion` | dense/list |
| Education, tutorial, learning | `chalkboard` | balanced/dense |
| Notes, handwritten, study | `study-notes` | dense/list/mindmap |
| Movie, poster, editorial, dramatic | `screen-print` | sparse/comparison |

## Outline Strategies

**Strategy A: Story-Driven** - 个人体验、情感共鸣。适合测评、个人分享。
**Strategy B: Information-Dense** - 价值优先、高效传递。适合教程、对比、清单。
**Strategy C: Visual-First** - 视觉冲击、精简文字。适合美学产品、生活方式。
